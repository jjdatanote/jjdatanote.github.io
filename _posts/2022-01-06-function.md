# [Lab] 계산기 함수
* 연산자, 두 수를 받아서 덧셈, 뺄셈, 곱셈, 나눗셈을 수행하는 함수


```python
def calc(op, n1, n2) :
    result = 0
    
    if op == '+' :
        result = n1 + n2 
    elif op == '-':
        result = n1 - n2
    elif op == '*':
        result = n1 * n2
    elif op == '/':
        result = n1 / n2
    
    return result 

print(calc('+', 8, 5))
print(calc('-', 8, 5))
print(calc('*', 8, 5))
print(calc('/', 8, 5))
```

    13
    3
    40
    1.6

# [Lab] 가변 인수의 평균값 계산
* 가변 매개변수로 들어오는 모든 수의 평균값 계산


```python
def avg(*args) :
    sum = 0
    for i in args :
        sum += i 
    return (sum / len(args))

print(avg(1, 2))
print(avg(1, 2, 3, 4, 5))
```

    1.5
    3.0
